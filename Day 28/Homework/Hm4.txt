დეკლარაცია არის CSS კოდის ის ნაწილი, 
რომელიც განსაზღვრავს, თუ როგორ უნდა გამოისახოს ელემენტი. 
დეკლარაცია შეიცავს property-ს (თვისებას) და value-ს (მნიშვნელობას), რომლებიც გამოყოფილია ორ წერტილით (:). 


CSS-ში კუთვნილება (property) არის სტილის სპეციფიური მახასიათებელი, რომელიც განსაზღვრავს, როგორ უნდა ჩანდეს ელემენტი. მაგალითად:

color განსაზღვრავს ტექსტის ფერს.
font-size განსაზღვრავს ტექსტის ზომას.
background-color განსაზღვრავს ფონური ფერს.


მნიშვნელობა არის კონკრეტული მნიშვნელობა, რომელიც კუთვნილებას ენიჭება. მაგალითად, red, 16px, center და სხვა.
მაგალითი:


CSS წესი არის მთლიანობაში ის კოდი, რომელიც სტილებს ანიჭებს HTML ელემენტებს. ის შედგება selector-სგან და დეკლარაციების ბლოკისგან.


სელექტორი არის ის ნაწილი, რომელიც განსაზღვრავს, რომელ HTML ელემენტზე მოქმედებს CSS წესი.


CSS მემკვიდრეობა გულისხმობს, რომ ზოგიერთ თვისებას (მაგალითად, color ან font-family) შვილი ელემენტები მემკვიდრეობით იღებენ მშობელი ელემენტისგან.



შვილობილი სელექტორი არის სელექტორი, რომელიც გამოიყენება იმ ელემენტების სტილის შესაცვლელად, რომლებიც სხვა ელემენტების შიგნით არიან.
